# 简介

<!-- 支持轻量化快速接入的 IoT 设备统一接入平台 -->

**寒霜物联 (Frost IoT)** 是一款开源的物联网设备统一接入平台。



与其他的物联网接入平台不同，寒霜物联项目从设计之初就考虑了复杂网络工况，采用 `adapter <-> gossip <-> sink` 模块设计。`adapter` 和 `sink` 节点支持根据实际需要进行扩展。节点之间支持通过多种通信协议进行通信。

- `adapter` 节点用于对接不同类型的硬件设备，并将其数据报文传递给后续节点。
- `gossip` 节点用于逐级传递设备报文数据，支持跨多内、外网进行传递。
- `sink` 节点用于解析设备报文，并将报文数据持久化
